NBA Offensive Statistical Dashboard

For this project, I conducted an in-depth analysis of evolving offensive trends in the NBA over the past 20 seasons. The focus was on examining how playing styles have changed, including the impact of the three-point shot, the perception of a "softer" NBA, and the decline in defensive play. I used web scraping techniques to gather the necessary data with BeautifulSoup and Requests to scrape 20 tables of data from Basketball-Reference for each NBA season. The scraped data, which included league averages for NBA offenses, was stored in a Pandas DataFrame for further processing. I then visualized the findings using Power BI, creating a dashboard that presented the trends and allowed for comparative analysis of offensive statistics over the years. This project provided valuable insights into how the NBA's offensive strategies and overall gameplay have evolved over the past two decades.

Skills: Python, Pandas, BeautifulSoup, Power BI, Web Scraping, Data Extraction, Data Cleaning, Data Transformation, Data Visualization, Statistical Analysis, Trend Analysis

NFL Statistical Analysis

I scraped offensive statistics for all 32 NFL teams spanning from 2013 to 2023, conducting an in-depth analysis of offensive trends, correlations, and team comparisons to derive actionable insights. I obtained the data from the NFL website using BeautifulSoup and Requests, successfully extracting 22 tables of data—11 for passing statistics and 11 for rushing statistics—covering the entire 11-year period. To store and process the data efficiently, I used Pandas to organize it into structured DataFrames. I then used Matplotlib to visualize key trends, making it easier to compare team performances and identify correlations between offensive metrics. My analyses included examining offensive trends from 2013 to 2023, identifying relationships between key statistics, comparing team performances over time, and validating common narratives such as whether the NFL has evolved into a pass-first league and if offensive production has declined in recent years.

Skills: Python, Pandas, Matplotlib, BeautifulSoup, Requests, Data Cleaning, Web Scraping, Data Extraction, Data Transformation, Data Visualization, Statistical Analysis

Nintendo Switch Game Search Engine

This project is a game search tool that scrapes data from MobyGames to retrieve information on Nintendo Switch game titles. Using web scraping techniques with BeautifulSoup, it collects game details from multiple pages of the website. The project then processes and cleans the data to ensure accuracy. This approach helps retrieve the most relevant game results based on textual similarity.
